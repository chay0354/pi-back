# Profile pictures bucket

Profile pictures (user creation / subscription) are stored in Supabase Storage bucket **`profile-pics`**.

## Create the bucket (if it doesn't exist)

1. In **Supabase Dashboard** go to **Storage**.
2. Click **New bucket**.
3. Name: `profile-pics`.
4. Set **Public bucket** to **Yes** so profile image URLs work in the app.
5. Create the bucket.

No SQL is required; storage buckets are created in the dashboard.
